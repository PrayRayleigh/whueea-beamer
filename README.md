# 武汉大学遥感信息工程学院Beamer模板

## 使用说明
本模板改编自项目[coventry_beamer](https://github.com/dscroft/coventry_beamer),编译方式为`xelatex -> biber -> xelatex*2`。模板内置有6种自定义颜色和3种页面比例对应背景水印排布方式，请自行根据需求修改文件使用。
